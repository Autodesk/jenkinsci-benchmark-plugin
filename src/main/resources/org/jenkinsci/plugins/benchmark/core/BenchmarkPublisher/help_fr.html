Le plugin de 'Comparaison de résultats' assemble and présente les résultats de type numériques et digital. Les résultats peuvent venir de sources multiples. Les résultats peuvent être au format XML ou JSON. Leurs contenus peuvent suivre un schéma standard ou personnalisé. Le type de schéma peut être sélectionné au travers du menu déroulant et le contenu d'un schéma personnalisé introduit à l'aide de la bande de texte associée.

De plus, le plugin de 'Comparaison de résultats' a la capacité d'ajouter des tests de validation supplémentaires sur les valeurs numériques. Si un résultat ne remplit pas ces contraintes, la résultat ainsi que la t\u00E2che principale recevra une condition d'échec.

Le plugin de 'Comparaison de résultats' fourni deux pages de résultats. Une avec deux tables de comparaison pour les données de résultats pures et condensées. Une autre avec les détails d'un résultat sélectionné.
